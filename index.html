So How Do You Set Up a Small Network?

So you want to set up a small network for your business or house... Well then you have come to the right place

So what exactly do you need?

Well first things first you need some hardware to actually set it up in the first place. Most of the stuff in this list is required but there is some extras you can buy, for example a bridge to connect this new network to an already existing network.

  • One or more switches/hubs. You need this to link your devices to each other.
  • Network cables so that you can connect all your devices to the aforementioned hub or switch.
  • You would also need a seperate wireless access point (but this could be a part of your switch). This is needed to allow your wireless devices (for example phones and laptops) to join the network you are creating.
  • You would need a router to connect the LAN you are making to the general internet (which is a WAN)
  • A firewall would also be required so that you can protect your network from hackers
  • You would also want to acquire a couple of small scale servers so that you can manage network security, network file storage, shared resources (this could include printers and scanners)

Do I need anything else?

Yes, you do need to organise some other things that is not just buying hardware.
You would need to do the following

  • Set up an account with an Internet Service Provider (ISP) of your choice (Note: This step only applies if you have not already got an internet connection for your location)
  • You would need to get an Internet connection installed from the ISP to your location (Note: This step only applies if you have not already got an internet connection for your location)
  • You may also need to configure various bits of hardware and software so that everything will work smoothly within the network
